#### 软件架构
开发板资源说明
• 支持Bluetooth 5.1的单模低功耗蓝牙SoC
• 多功能按键和LED指示灯
• 支持Arduino模块插接接口，IO电压可以通过level shift灵活配置
• 支持调试功能的SEGGER J-Link OB
• UART转USB接口
• Micro USB接口连接PC
• 1.44寸TFT彩色显示屏 
• 板上集成QSPI Flas
