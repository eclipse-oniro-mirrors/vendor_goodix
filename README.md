# vendor_goodix

## 介绍

该仓库托管Goodix开发的基于GR551x BLE产品样例代码。

## 软件架构

支持基于Goodix GR5515 SoC的Starter Kit单板进行开发的XTS测试样例。

代码路径：

```
device/                               --- 硬件单板相关仓库
├── boards/goodix                     --- Goodix相关单板
├── soc/goodix                        --- Goodix相关SoC代码
vendor/goodix/                        --- vendor_goodix 仓库路径
└── gr5515_sk_xts_demo                --- XTS测试样例
```
